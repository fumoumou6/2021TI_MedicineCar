# 2021TI_MedicineCar
2021电赛F题，自动送药小车

## 一.设计思路：
### 1.硬件电路  
#### （1）识别运算平台：

    树莓派4b
#### （2）运动控制平台：

    openmv4H7Plus
#### （3）电机驱动模块：

    淘宝正常的电机驱动模块，驱动芯片为L289n
#### （4）车身：

    淘宝正常符合电赛要求的车身

## 二.功能分类：
### （1）树莓派：
    通过调用百度智能云OCR的api进行文本识别，识别率99%，单张图片识别时长1~4秒(根据网络情况而定)
    
    百度OCR详细地址： https://cloud.baidu.com/product/ocr/general (大厂的东西就是好用)
